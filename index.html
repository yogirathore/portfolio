ABOUT THE ORGANIZATION

The Delhi metro doesn't have an app to know your last travel history through delhi metro, so I think this is a great opportunity to create a UI design for the metro. By having a record of your travel history, you'll be able to see patterns and optimize your trips. The UI would need to be simple and easy to use, since many people use the metro on a daily basis.

PHASE 1

WIREFRAME

Creating a wireframe for a UI design is much like sketching out the blueprint for a building. You need to start with the bare bones structure and then add in the details as you go. For my design of the DMRC, I started by creating a basic layout of where all the elements would go.

CASE STUDY

CHOICES & CONCEQUENCES

LIVE PREVIEW
ABOUT THE ORGANIZATION

I got the freelance work to create moblie ui design for a company called CHOICES & CONCEQUENCES. The project was to create a design for their new app which helps people make better choices regarding their health and well-being. It was great working with such a passionate and dedicated team, and I'm really happy with the end result.

PHASE 1

WIREFRAME

I created a UI wireframe for choice and consequences that is easy to use. The purpose of this wireframe is to help you make decisions quickly and easily, without having to think about the consequences too much. To do this, I used a simple layout with clear labels and buttons. I also made sure to include plenty of space for feedback so you can see the results of your choices immediately.
